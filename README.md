<h1 align="center">catto</h1>
Get a cat from https://cataas.com and show the cat in the terminal using chafa

## requirements

- libcurl-devel
- chafa
- argp-standalone (macos only)

## instructions

- build with build.sh
- install with install.sh
- type `catto` (`-h` for help).
- enjoy your pretty kitty!
